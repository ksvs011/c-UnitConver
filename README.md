# c-UnitConver

### 개요
- 임의의 숫자를 입력해 cm -> inch, inch - > cm로 단위변환 하는 프로그램
- 자유롭게 메뉴를 이동 가능
- 'cls'를 사용하여 콘솔창을 깔끔하게 정리
